
This is a small project that I use to test the python library pytorch.

# Usage

To install required packages, install `nix` and use `nix develop -c bash`.
You can replace `bash` with any other shell (zsh, 42sh, ...).
It will also install python packages.

**Run the project:**
- `pytest` will train small models and check that they work
- `python src/main.py` will train bigger models, listed in `main.py`,
  if they are not already trained

**View the results:** use `tensorboard`. Ex: to see results of `mnist_balanced`
runned with 1500 epochs, run `tensorboard --logdir ./runs/mnist_balanced_1500`.

# Project tree

- `src`: manage user interaction, might contain notebooks
- `src/lib`: contains general classes and functions,
  without direct user interaction, like a lib
- `tests`: contains unit and functional tests, if needed

The folders `runs`, `data` and `models` are generated by the program and
can be safely removed if you want to train everything again.

# Installed python packages

- `torch` for Machine Learning
- `torchvision` for datasets
- `matplotlib` for visualisation
- `pytest` & `pytest-sugar` & `pytest-xdist` for unit tests
- `tqdm` & `rich` for logs and stdout

Note that tensorboard will not show results generated with `pytest` if multiple
process are used using `pytest-xdist`.
